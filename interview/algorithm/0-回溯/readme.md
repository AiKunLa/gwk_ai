# 回溯算法

深度优先搜索（DFS）：回溯通常采用 DFS 的方式遍历解空间树。

递归实现：回溯算法天然适合用递归来实现，因为每一步的“选择 → 探索 → 撤销”结构与递归调用栈高度契合。

剪枝优化：在搜索过程中，若能提前判断某条路径不可能产生有效解，就跳过该分支，称为“剪枝”，可大幅提升效率。

## 模板

def backtrack(当前状态):
    if 满足结束条件:
        将当前解加入结果集
        return
    
    for 可选选择 in 所有可能的选择:
        做出选择
        backtrack(新状态)
        撤销选择  # 回溯的关键步骤

## 定义状态
明确 dp[i] 或 dp[i][j]代表什么含义

一般是用于存放子问题的结果，父问题需要用到子问题的结果。或者收 如果我知道前面的结果，如何推出当前位置的结果
eg:
    dp[i]: 以第i给元素结尾的最大和
    dp[i][j]: 字符串 s1[0...i-1]和 s2[0...j-1]的最长公共子序列长度


## 状态转移方程

找出当前状态与之前转台的关系
eg:
    dp[i] = max(dp[i-1] + num[i], nums[i]) 最大子数组和
    dp[i][j] = dp[i-1][j-1] + 1     LCS
    dp[i][j] = min(上，左，左上) + 1 最大正方形


## 初始化


设置边界条件，避免越界或错误起点
通常处理 i = 0 j = 0
示例：
    dp[0] = nums[0]
    dp[i][0] = 0
    dp[0][j] = j


## 确定遍历顺序

决定如何填充 DP 表。

一维：从左到右 / 从右到左
二维：
行优先（for i in range(m): for j in range(n)）
列优先
对角线（区间 DP）
注意依赖方向：必须先计算被依赖的状态！


## 常见DP模板

线性DP
    状态定义特点 ：dp[i] 依赖 dp[i-1], dp[i-2]
    题目：爬楼梯、打家劫舍、最大子数组

二维坐标 DP
    dp[i][j] 表示走到 (i,j) 的最优值
    最大正方形、路径最小和

背包 DP
    dp[i][w] 表示前 i 个物品、容量 w 的最大价值
    0-1 背包、完全背包

区间 DP
    dp[i][j] 表示区间 [i,j] 的最优解
    石子合并、回文子串计数

状态压缩 DP
    用位掩码表示状态
    N皇后、铺瓷砖

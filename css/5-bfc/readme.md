  - container原本是block 块级盒子   现在升级为BFC盒子。

  - BFC 盒子的特征
    - 这是一个全新的渲染区域，不受外界的影响

## 什么是BFC
BFC 是 Web 页面中的一个独立的渲染区域，它规定了内部的块级盒子如何布局，并且与外部区域毫不相干。
